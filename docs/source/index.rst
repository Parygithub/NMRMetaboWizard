NMRMetaboWizard documentation
==============================

NMRMetaboWizard is an interactive Shiny for Python application for stepwise
preprocessing and analysis of one-dimensional Bruker 1H NMR metabolomics data.

.. warning::

   NMRMetaboWizard is a research tool. It is not a clinical diagnostic system,
   medical device, or substitute for expert review.
