# Availability and implementation

## Bioinformatics-style statement

**Availability and implementation:** NMRMetaboWizard is implemented in Python
using Shiny for Python, Plotly, NumPy, pandas, SciPy, scikit-learn, Matplotlib,
openpyxl, and shinywidgets. Source code is available under the MIT License at
https://github.com/Parygithub/NMRMetaboWizard. User documentation is available at
https://nmrmetabowizard.readthedocs.io. The version used in this study is archived at
https://doi.org/10.5281/zenodo.REPLACE_WITH_RECORD. The software requires Python 3.10 or newer and is
platform independent; Chrome or Microsoft Edge is recommended for the
interactive interface.

## Data availability

The source code and synthetic metadata template are publicly available in the
software repository. Raw clinical NMR data are subject to ethics approvals,
participant consent, institutional governance, and applicable privacy
regulations. A de-identified or controlled-access data statement should be
inserted here according to the final sharing arrangement.

## Code availability

The exact release used for the manuscript should be identified by version tag
and Zenodo DOI. Analysis outputs required to reproduce the manuscript figures
