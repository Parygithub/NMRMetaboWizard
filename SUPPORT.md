# Support

Use GitHub Issues for:

- installation problems;
- reproducible software errors;
- documentation corrections;
- feature requests.

A good issue includes:

- NMRMetaboWizard version;
- operating system;
- Python version;
- browser;
- the complete error traceback;
- a synthetic or non-confidential example;
- the preprocessing or analysis step where the error occurred.

The maintainers cannot provide clinical interpretation or patient-specific diagnostic advice.
